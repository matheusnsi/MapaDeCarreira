# Mapa de Carreira — Matheus Nascimento Silveira

Mapa de carreira e mini-portfólio profissional desenvolvido como parte do programa
**Instituto PROA** (em parceria com o **Senac**). A página apresenta minha trajetória,
meus objetivos de carreira e minhas habilidades como desenvolvedor.

🔗 **Acesse online:** https://matheusnsi.github.io/MapaDeCarreira/

---

## 📌 Sobre

Sou **Matheus Nascimento Silveira**, desenvolvedor full-stack em formação, com base em
front-end (React, TypeScript) e foco de crescimento no back-end (C#/.NET). Atuo como
desenvolvedor estagiário na **Multitools**, onde, ao lado de outro estagiário, sou
responsável por todo o desenvolvimento de sistemas da empresa.

Este projeto reúne, em uma única página, minha apresentação, meu planejamento de carreira
de curto, médio e longo prazo, minhas competências técnicas e meu currículo para download.

## 🧭 Seções da página

- **Prazer em conhecer** — apresentação e trajetória
- **Mapa de carreira** — plano de evolução em três fases (Júnior → Pleno → Sênior/Liderança),
  com objetivos, soft skills e roadmap de aprendizado de cada etapa
- **Habilidades** — tecnologias com que trabalho e as que estou aprofundando
- **Idiomas**
- **Currículo** — disponível para download em PDF

## 💻 Como visualizar localmente

Por ser um site estático, basta abrir o `index.html` no navegador.
Para evitar restrições de carregamento de arquivos, recomendo um servidor local simples:

```bash
# com Python 3
python -m http.server 8080
```

Depois acesse `http://localhost:8080`.
